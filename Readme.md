# x-monitor

![x-monitor](./doc/img/x-monitor.jpg)

观察系统、应用、内核的运行状态，从以下几个方面：

- 传统的系统运行指标

- 应用运行的指标

- CGroups 状态指标

- eBPF 内核状态指标

- 进程的Profiling，CPU Flame Graphs
